Guide d'utilisation du DVD-ROM

Le mode d'emploi de ce DVD-ROM et des cours multimédia est disponible ici.




Configuration matérielle

Pour visualiser les cours multimédia et les documents PDF présents sur le DVD-ROM, vous devez disposer :
  • D'un ordinateur multimédia sous Windows ou macOS
  • Equipé d'un lecteur de DVD-ROM
  • Equipé d'une carte son connectée à des hauts-parleurs ou à un casque audio

Configuration logicielle

La configuration logicielle nécessaire pour la lecture des cours multimédia est la suivante :
  • un navigateur Internet (dernières versions de Mozilla Firefox ou Google Chrome recommandées)
  • le lecteur Adobe Acrobat Reader sous forme de plug-in pour votre navigateur Internet est nécessaire pour la lecture des documents PDF présents sur le DVD-ROM.

Installation du lecteur de documents PDF Adobe Acrobat Reader
L'installation du plug-in Adobe Acrobat Reader est le plus souvent gérée automatiquement par votre navigateur Internet lorsque vous êtes en ligne mais la dernière version du programme d'installation générique est téléchargeable gratuitement sur le site d'Adobe en cliquant ici.



Test de votre ordinateur

Pour tester votre ordinateur vous pouvez accéder à une présentation multimédia en cliquant ici. Si vous parvenez à visualiser cette présentation multimédia votre ordinateur est parfaitement configuré pour étudier.

Si vous rencontrez des problèmes pour lire cette présentation consultez le paragraphe ci-dessous sur les problèmes techniques connus et leurs solutions.

Accès à la plateforme d'enseignement en ligne

Le site Internet de la Première Année des Etudes de Santé (PACES) de l'Université Grenoble Alpes est disponible à l'adresse http://paces.medatice.fr

Vous disposez sur ce site d'un espace personnel de formation qui vous permet d'accéder :
  • aux cours multimédia présents sur ce DVD-ROM ainsi qu'à d'éventuels documents complémentaires
  • à la Formulation en Ligne des Questions (FLQ) afin de rédiger des questions pour les enseignants avant les Séances d'Enseignement Présentiel Interactif (SEPI) : cliquer ici
  • aux annales des tutorats, aux épreuves des tutorats ainsi qu'à vos résultats des tutorats (notes et classements) : cliquer ici
Vos identifiants universitaires personnels de connexion AGALAN sont nécessaires pour accéder à votre espace personnel de formation. Ces identifiants vous sont fournis par l'Université Grenoble Alpes lors de votre inscription définitive.
Problèmes techniques connus et solutions
En cas de problème : il est recommandé d'utiliser la dernière version du navigateur Mozilla Firefox ou Google Chrome pour consulter le DVD.

Voici la liste des problèmes le plus fréquement rencontrés et leurs solutions :

Rien ne se passe lorsque je place le DVD-ROM dans le lecteur de DVD de mon ordinateur
  • Problème : Vous devez au préalable vérifier que le lecteur de votre ordinateur est capable de lire des DVD-ROM et pas seulement des CD-ROM en consultant la documentation de votre ordinateur. Le contenu du DVD-ROM n'est pas lisible sur les platines vidéo de salon.
  • Solution : Il faut ouvrir manuellement le fichier index.htm présent à la racine du DVD-ROM en double cliquant dessus. Sur PC vous pouvez cliquer sur démarrer puis Poste de travail puis dans la liste de droite cliquez sur votre lecteur de DVD-ROM PACES et enfin ouvrez le fichier index.htm en faisant un double-clic desssus ou bien en faisant un simple-clic gauche puis simple-clic droit et enfin cliquer sur ouvrir dans le menu contextuel. Le navigateur Internet installé sur votre ordinateur doit alors se lancer et afficher le contenu du fichier index.htm.
  • Astuce : afin de consulter en silence et plus rapidement vos cours, il est conseillé de copier l'ensemble des fichiers présents sur le DVD-ROM vers le disque dur de votre ordinateur puis de consulter vos cours depuis votre disque dur plutôt que depuis le DVD-ROM.
Le cours multimédia ne s'affichent pas dans mon navigateur Internet
  • Solutions : Installer ou réinstaller la dernière version du navigateur Mozilla Firefox ou Google Chrome
  • Si le problème persiste : Changer de navigateur Internet en prérant les dernières versions de Mozilla Firefox ou Google Chrome.
Je n'ai pas de son lors de la lecture du cours multimédia : si le message Pas de son s'affiche dans la barre du lecteur




  • Si le message pas de son est présent sur la première ET la deuxième diapositives : rechargez simplement la page en cliquant sur la flèche de réactualisation de votre navigateur et le son devrait démarrer. Il est parfois nécessaire de recharger plusieurs fois la page pour que le son soit chargé dans le lecteur. Il est important de laisser du temps au navigateur entre 2 réactualisations.
  • Si le problème persiste : téléchargez la dernière version de votre navigateur ou changez de navigateur en préférant les dernières versions de Mozilla Firefox ou Google Chrome.
Le cours multimédia s'arrête à la fin d'une diapositive sans enchainer automatiquement sur la suivante :




  • Solutions : Vous devez relancer manuellement la diapositive suivante en cliquant sur son titre dans le menu de navigation ou en cliquant sur le bouton suivant présent dans la barre de navigation du bas de page.
Je n'arrive pas à lire les documents joints au format PDF
  • Problème : Le plug-in Adobe Acrobat Reader n'est pas installé pour votre navigateur ou est mal installé.
  • Solutions : Installer ou réinstaller le plug-in Adobe Acrobat Reader :
    Installer le Plugin Adobe Acrobat Reader : Vous pouvez installer le plugin en téléchargeant la dernière version du programme d'installation à partir du site d'Adobe en cliquant ici.
    Si le problème persiste : téléchargez la dernière version de votre navigateur ou changez de navigateur en préférant les dernières versions de Mozilla Firefox ou Google Chrome.
    Si le problème persiste encore : vous devez désinstaller l'ensemble des versions du lecteur Adobe Acrobat Reader installées sur votre ordinateur puis réinstaller le lecteur Adobe Acrobat Reader à partir du site d'adobe. Une page dédiée à la désinstallation du plugin Adobe Acrobat Reader est présente sur le site d'Adobe en cliquant ici.
